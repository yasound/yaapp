Tadaam ! La version Web de YaSound est prête ! Et pour découvrir le nouveau visage de YaSound dans les meilleures conditions nous vous offrons un mois d’abonnement premium pour écouter toutes les radios en haute qualité.

Pour en profiter rien de plus simple, il vous suffit de vous rendre sur le site pour créer votre compte et vous serez crédité d’un mois de HD.

Si vous possédez déjà un compte ne vous inquiétez pas, celui-ci a été automatiquement crédité d’un mois d’abonnement gratuit.

  • Il est maintenant possible de créer plusieurs radios pour un même compte.
  • La programmation de votre radio peut se faire à partir des titres de vos playlists Deezer.
